Notes of CSS

Style

  • Style tag for adding style.
  • Under style section, there would be separate style for each tag we want to change.
  • For example, If we want to put style on paragraph or on h1 section, we should only write p{style} or h1{style}.
  • Back-ground Color, Color, font size, font-weight, height, width, font-family or are styles.

Class & ID

If we do the aforementioned tactics, then the style will be fall for each time we use p tag or h1 tag. But If we want to put style on a single paragraph or single h1 section then we should use ID. for using ID we just have to put a # before id name.

If we want to use same style for several tag but not all, then we can use Class. Use of ID and Class is quite same. The difference is for class, we have to put a dot (.) instead of hash(#)

Div & Span

DIV and SPAN. For formatting a small portion of text we use span.

Border

  • We can add border by border tag.
  • border-radius: to add curve

Margin

To add margin - top, right, bottom, left. Clockwise.

Padding

To add padding: padding -top, right, bottom, left.

The difference between padding and margin is: margin adds space to outside and padding adds to inside.

CSS BOX Model

Inline & Block & Inline-Block (Display)

  • span, small, strong, b, i these are inline.
  • div, p, li these are block.
  • nline-block display: combination of inline and block level display. Suppose we want to make a list but instead of showing it vertically, inline-block display will show it vertically.
